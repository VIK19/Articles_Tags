Описание инсталляции и программ сайта Articles Catalog

1.Инсталляция БД
Структура БД описана в файле doc/DB.txt.
Таблицы создавать в порядке их перечисления в doc/DB.txt.

2.Основной файл сайта - index.php

Примечание.
Функции:
- подтверждение пароля;
- изменение пароля;
- ввод URL-адреса
НЕ ПРОТЕСТИРОВАНЫ, Т.К. они требуют обратную связь через E-mail
(в DENVER не удалось это сделать).
Для работы зарегистрированному пользователю нужно чтобы в поле 
users_status таблицы users была установлена 1 вместо 0.

3.Описание папок и файлов.
3.1.В папке css размещен файл таблицы стилей - main.css
3.2.В папке doc   3 файла:
DB.txt - структура БД
Install_Article_Site.txt - описание инсталляции и программ сайта Articles
/doc/scheme_site.txt - упрощенная схема файлов сайта и их краткое описание.
3.3.В папке img расположены файлы изображений для пиктограмм.
3.4.В папке js приведены стандартные и собственная библиотеки js-программ.
3.5.В папке txt находятся файлы фраз меню для 2-х языков - английского и руского.
Примечание.
По умолчанию на сайте установлен английский язык.
